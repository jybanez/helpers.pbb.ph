Tab Switching

Switch tabs and watch change events emit into the local log.

createTabs(host, {
  activeId: "one",
  tabs: [
    { id: "one", label: "Tab One", content: "Tab one content." },
    { id: "two", label: "Tab Two", content: "Tab two content." },
    { id: "three", label: "Tab Three", content: "Tab three content." },
  ],
  onChange(tab, id) {
    writeLog("tabs.change", { id, label: tab?.label });
  },
});
