Modal Demo

Dedicated shell demo for free-form modal content, owner-title context, draggable headers, footer content, and explicit lifecycle logging.

Reusable Modal Shell

Open either example to inspect the shared shell. The reusable example shows the owner-title line and default header dragging. The long-content example keeps the header pinned at the top and the footer pinned at the bottom while only the body scrolls.

const modal = createModal({
  title: "Reusable Modal",
  ownerTitle: "PBB HQ",
  draggable: true,
  size: "md",
  content: body,
  headerActions: [headerRefreshBtn, headerHelpBtn],
  footer,
  onClose(meta) {
    writeLog("modal.close", meta || {});
    modal.destroy();
  },
});

modal.open();
const longModal = createModal({
  title: "Long Content Modal",
  size: "md",
  content: longBody,
  footer,
});

longModal.open();
// header stays fixed
// footer stays fixed
// only .ui-modal-body scrolls
