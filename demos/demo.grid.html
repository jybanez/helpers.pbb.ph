Grid Demo

Dedicated playground for local, remote, and large virtualized grids.

Use This When

Use this when

Use createGrid(...) for flat tabular data where search, sort, pagination, remote query flows, or dense action cells matter.

Do not use this when

Do not use it for true hierarchy. If the first column represents parent-child structure, use createTreeGrid(...) or createHierarchyMap(...) instead.

Local Grid

Client-side search, sort, pagination, and selectable rows.

Remote Grid

Grid emits query changes; parent simulates API paging and sets rows.

Large Grid (Virtualized)

Fixed-height scroll container with virtualization enabled for large row count.
