Number Stepper Demo

Dedicated helper page for the shared numeric stepper used for counters, prices, and bounded quantity input.

Currency-Style Stepper

This sample mirrors the plus/minus price-style control: typed edits commit on blur or Enter, while buttons step immediately within the configured range.

const stepper = createNumberStepper(host, {
  value: 1,
  min: 0,
  max: 10,
  step: 0.25,
  decimals: 2,
  prefixText: "$",
  onChange(value) {
    writeLog("stepper.change", { value });
  },
});
