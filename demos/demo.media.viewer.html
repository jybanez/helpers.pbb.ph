Media Viewer Demo

Standalone image/video viewer with navigation, zoom, pan, fit modes, and optional video audiograph.

Use This When

Use this when

Use createMediaViewer(...) when image or video review needs zoom, pan, fit control, gallery navigation, or a viewer that stands on its own outside a strip.

Do not use this when

Do not use it when you only need thumbnail launching. Use createMediaStrip(...) for thumbnail surfaces and let it delegate full-view behavior to the viewer.

Keyboard: Esc close, ←/→ navigate, +/- zoom, 0 reset, Home/End jump.

Media Items

Single-Item Repro

Use this section to compare the website-reported single-item case against a repeated-item control case without changing the viewer instance. Both actions reuse the same createMediaViewer(...) instance.

const singleItem = [firstImage];
viewer.update({ items: singleItem, index: 0, fit: "contain", showFooter: false, baseUrl: mediaBaseUrl });
viewer.open(0);
