Build And Opt-In

The source tree stays modular. The build step just produces a minified bundle for downstream apps that want fewer helper requests.

npm run build:ui-bundle

import { createUiLoader, DEFAULT_COMPONENT_REGISTRY } from "../js/ui/ui.loader.js";

const bundledLoader = createUiLoader(DEFAULT_COMPONENT_REGISTRY, {
  preferBundles: true,
});

const createFormModal = await bundledLoader.get("ui.form.modal");

For clean request-count comparisons in a real app, refresh the page between modular and bundled runs so previously loaded CSS/modules do not stay warm in memory.
