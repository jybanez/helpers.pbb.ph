Window Manager Demo

Dedicated demo for the shared desktop-style window manager. This page is intentionally workspace-oriented: it shows how the manager behaves when several windows stay open and coordinated in one surface.

Incident Review Workspace

Wrapper demo over ui.window + ui.data.inspector. This is the kind of non-modal analyst surface that benefits from stacked windows instead of modal churn.

function openIncidentWorkspace(incident) {
  ensureWindow(incidentManager, incidentWindows, "incident-summary", () =>
    incidentManager.createWindow({
      title: "Incident Summary",
      content: buildIncidentSummaryContent(incident),
    })
  );

  ensureWindow(incidentManager, incidentWindows, "incident-payload", () =>
    incidentManager.createWindow({
      title: "Incident Payload",
      content: buildIncidentPayloadContent(incident),
    })
  );
}
