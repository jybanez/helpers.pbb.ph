Datepicker Demo

Dedicated picker demo for single-date and range-plus-time selection behavior.

Operational Date Capture

The single picker is suited for one incident date; the range picker models active windows or maintenance periods.

Single Date
Range + Time 
createDatepicker(document.getElementById("datePickerHostSingle"), {
  ariaLabel: "Incident date picker",
  placeholder: "Pick incident date",
  mode: "single",
  onChange(value, state) {
    writeLog("datepicker.single.change", { value, state });
  },
});

createDatepicker(document.getElementById("datePickerHostRange"), {
  ariaLabel: "Incident date range picker",
  placeholder: "Pick active range",
  mode: "range",
  showTime: true,
  closeOnSelect: false,
  onChange(value, state) {
    writeLog("datepicker.range.change", { value, state });
  },
});
