Game Core Controls Demo

Switch between the D-pad and virtual joystick. Use either control or keyboard arrow keys to move the marker.

Embedded Session Preview

Usage

const session = createGameSession(host, {
  title: "Canvas Game",
  fullscreen: true,
  width: 720,
  height: 720,
});

const layer = session.addLayer({ id: "board" });
const controls = document.createElement("div");
session.overlay.appendChild(controls);

createVirtualJoystick(controls, {
  deadzone: 0.12,
  onMove(value) {
    player.vx = value.x * speed;
    player.vy = value.y * speed;
  },
});
