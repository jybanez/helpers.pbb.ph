Drawers Demo

Dedicated anchored-drawer demo with composed search, chip content, and declarative header actions.

Drawer Positions

Each button opens the same drawer pattern from a different side of the viewport.

const drawer = createDrawer({
  title: "Users",
  position: "right",
  headerActions: [{
    id: "refresh",
    label: "Refresh users",
    icon: "actions.refresh",
    onClick(action, event, drawerApi) {
      refreshUsers();
    },
  }],
});
drawer.open(document.body);
