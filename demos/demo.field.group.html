Metadata-only Preset

This mirrors backend-provided incident type fields where the database stores the preset metadata, not a duplicated fields[] schema.

const metadataOnly = createFieldGroup(host, {
  field_key: "missing_persons",
  field_label: "Missing Persons",
  input_type: "group",
  config_json: JSON.stringify({
    preset: "missingPerson",
    preset_label: "Missing Persons",
    repeatable: true
  })
});
