Window Composition

This section proves the intended composition: keep ui.window generic and host embedded content through createIframeHost(...).

const iframeHost = createIframeHost({ src: fixtureUrl, title: "Workspace Preview" });
const manager = createWindowManager({ container: surface });
manager.createWindow({ title: "Workspace Preview", content: iframeHost.root });
