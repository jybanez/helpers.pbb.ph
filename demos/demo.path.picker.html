Path Picker Demo

Local setup fields with app-owned picker adapters and helper-owned path validation/status UI.

The browser demo uses mock adapters; PBB Kit Setup should wire these buttons to its native/local-agent file and folder picker.

PBB Kit Platform Paths

Sample Code

const phpPicker = createPathPicker(host, {
  label: "PHP",
  mode: "file",
  extensions: [".exe"],
  browseLabel: "Choose PHP executable",
  pickFile(context) {
    return kitBridge.pickFile(context);
  },
  validatePath(path) {
    return kitBridge.validateExecutable(path);
  }
});
