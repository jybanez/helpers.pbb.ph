Change Password Form Modal Demo

Dedicated password-change preset with helper-owned current/new/confirm structure and shared password-toggle behavior.

Password Change Flow

const modal = createChangePasswordFormModal({
  fields: {
    currentPassword: "current_password",
    newPassword: "next_password",
    confirmPassword: "confirm_password",
  },
  async onSubmit(values, ctx) {
    if (values.next_password !== values.confirm_password) {
      ctx.setErrors({ confirm_password: "Confirmation password does not match." });
      return false;
    }
    return true;
  },
});
