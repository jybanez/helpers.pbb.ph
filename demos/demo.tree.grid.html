Tree Grid Demo

Hierarchical rows with first-column indentation and aligned succeeding columns.

Use This When

Use this when

Use createTreeGrid(...) when hierarchy needs aligned columns, expandable rows, tree-aware search, lazy loading, or virtualization.

Do not use this when

Do not use it for flat tables or purely visual hierarchy exploration. Use createGrid(...) for flat data and createHierarchyMap(...) for visual relationship reading.

Lazy Tree Grid (Chrome-less)

Rows marked with hasChildren: true load children only on expand. This instance also uses chrome: false.

Large Tree Grid (Virtualized)

Flattened visible rows are virtualized with fixed row height.
