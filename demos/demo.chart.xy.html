X/Y Chart Demo

Axis-based chart primitive for line trends, reference bands, and thresholds. Domain interpretation stays app-owned.

Point Events

Hover or select a point.

Usage

const chart = createXyChart(container, {
  xType: "date",
  xLabel: "Date",
  yLabel: "Minutes",
  series: [{ label: "Median", points: [{ x: "2026-07-01", y: 12 }] }],
  thresholds: [{ axis: "y", value: 15, label: "Target", tone: "warning" }],
  bands: [{ axis: "y", from: 0, to: 10, label: "Healthy", tone: "success" }],
});
