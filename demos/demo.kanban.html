Dedicated workflow-board demo for lane movement, WIP limits, rejection rules, and move callbacks.

Incident Workflow Board

Active lane has a WIP limit of three cards. Cards marked with review requirements cannot move directly into Resolved.

const board = createKanban(host, lanes, {
  ariaLabel: "Incident workflow board",
  wipLimits: { active: 3 },
  validateMove(payload) {
    if (payload.toLaneId === "resolved" && payload.card.requiresReview) {
      return { ok: false, reason: "Review required first." };
    }
    return true;
  },
  onCardMove(payload) {
    writeLog("kanban.card.move", payload);
  },
});
