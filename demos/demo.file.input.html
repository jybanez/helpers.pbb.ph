File Input Demo

Reusable file picker for selected, dropped, or pasted files. Upload behavior remains app-owned.

Picker Variants

Sample Code

const picker = createFileInput(host, {
  label: "Message attachments",
  accept: "image/*,video/*,audio/*,.pdf",
  multiple: true,
  preview: "icon",
  onChange(files, meta) {
    attachmentQueue.add(files, { source: meta.source });
  }
});
