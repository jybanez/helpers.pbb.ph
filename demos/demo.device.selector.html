Device Selector Demo

Browser media selectors below use the real browser APIs where available. The adapter example uses mock devices to show the future USB/Bluetooth/local-agent path without requiring hardware.

createDeviceSelector(host, {
  kind: "camera",
  label: "Camera",
}, {
  adapter: createMediaDeviceAdapter({ kind: "videoinput" }),
  presentation: "select",
  onSelectionChange({ selectedDeviceId }) {
    saveCameraId(selectedDeviceId);
  },
});
createDeviceSelector(host, {
  kind: "camera",
  devices,
  selectedDeviceId: "cam-2",
}, {
  autoRefresh: false,
  presentation: "menu",
});
