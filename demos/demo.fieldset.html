Fieldset Demo

Dedicated helper page for grouped form sections that can mix fields with text, warnings, images, and custom content using the same practical row vocabulary as form-modal bodies.

Workspace App Group

const fieldset = createFieldset(host, {
  legend: "Workspace App Settings",
  description: "Group operational app settings and supporting notes.",
  rows: [
    [
      { type: "input", name: "app_code", label: "App code" },
      { type: "input", name: "name", label: "Name" }
    ],
    [
      { type: "alert", tone: "info", span: 2, content: "Document the access-check endpoint before enabling the app in Workspace." }
    ],
    [
      { type: "image", span: 2, src: "../samples/media/photos/pbb-preview-graphic-2x.png", alt: "Workspace preview graphic", caption: "Supporting image/media rows can live inside the same grouped section." }
    ]
  ]
});
