Workspace Bridge Demo

Parent workspace host is installed on this page. The child app runs inside an iframe hosted by ui.window. Explicit bridge primitives still exist, plain helper modal-family overlays auto-route to the parent shell when the host is same-origin, and explicit cross-origin login/reauth form requests can use modal.form.open.

Window Composition

installWorkspaceUiBridgeHost();
const iframeHost = createIframeHost({
  src: "../samples/iframe/workspace-ui-bridge.fixture.html",
  title: "Workspace Bridge Child",
});
manager.createWindow({ title: "Bridge Child", content: iframeHost.root });
