Media Strip Demo

Dedicated thumbnail-strip demo for incident photos/videos, shared viewer delegation, viewer option syncing, and pending-media placeholders.

Thumbnail Browser

Both strip layouts share the same viewer options. Relative sample-media paths resolve against the repo base URL. The first card shows the new pending processing placeholder contract.

Wrap layout

const strip = createMediaStrip(host, items, {
  layout: "scroll",
  autoplay: true,
  baseUrl: mediaBaseUrl,
  viewerFit: mediaStripFitSelect.value,
  showViewerFooter: mediaStripFooterToggle.checked,
  showViewerToolbar: mediaStripToolbarToggle.checked,
  showViewerAudiograph: mediaStripAudiographToggle.checked,
  viewerAudiographStyle: mediaStripAudiographStyleSelect.value,
  onOpen(item, index) {
    writeLog("media.open", { index, id: item?.id, type: item?.type });
  },
});
