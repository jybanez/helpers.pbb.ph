Icon Grid Demo

Drag icons to reorder. Arrow keys move focus; Ctrl/Meta + Arrow moves the focused tile. The emitted layout is saved by the host app.

PBB Landing Launcher

Local Network CEBU CITY, CEBU pbb.ph

Usage

const savedLayout = JSON.parse(localStorage.getItem("pbb.launcher.layout") || "null");

const launcher = createIconGrid(host, apps, {
  columns: 3,
  slots: 6,
  chrome: false,
  autoArrange: false,
  scrollable: true,
  maxHeight: 360,
  touchDragDelay: 280,
  touchHoldTolerance: 36,
  layout: savedLayout,
  onLayoutChange(layout) {
    localStorage.setItem("pbb.launcher.layout", JSON.stringify(layout));
  },
  onActivate(item) {
    uiAlert(`Opening ${item.label}`, {
      title: "Launcher Icon Clicked",
      variant: "info",
      description: item.href,
    });
  },
});
