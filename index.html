Incident Helpers Library

This project contains modular JS/CSS helpers for incident UI components. Each helper accepts (container, data, options) and returns a stable API with destroy() and update(nextData, nextOptions).

How To Use

Import the helper you need, pass a target DOM element, render data, and behavior options. Viewer helpers render read-only output. Editor helpers emit callbacks for mutations.
